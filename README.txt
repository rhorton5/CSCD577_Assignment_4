Compile the folders separately using CMAKE.  All folders have their own bin and src folders.

CarotidFlowLocal folder is based problem 2 of the assignment.
This program will prompt you to enter an x, y, an z value before opening the window.  This is what will generate the velocity distrubtion and the sphere's location.


CarotidFlowDistributionZ will contain two files:
CartoidFlow_Problems_3_1 has the seed of (9,9,z) where you input what z will be.
CartoidFlow_Problems_3_2 has the seed of (6,6,z) where you input what z will be.
